## üåê Usage Instructions

After installation, you can start using ytdl-go to download videos. Here‚Äôs how to use it:

1. Open the Command Prompt (Windows) or Terminal (macOS/Linux).
2. Type the command `ytdl-go [options] <URL>` to download a video. Replace `<URL>` with the actual link of the YouTube video or playlist you want to download.
3. Options will help you customize your download. Here are some common options:
   - `--audio`: to download audio only.
   - `--video`: to download video only.
   - `--playlist`: to download an entire playlist.

### Example Command

To download a single video:
```
ytdl-go https://www.youtube.com/watch?v=example
```

To download a playlist:
```
ytdl-go --playlist https://www.youtube.com/playlist?list=example
```
